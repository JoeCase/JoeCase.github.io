GIT COMMANDS

git init create new git repository
git checkout master move to local master branch
git pull (origin master) pull remote
git branch [new-branch-name] create new branch
git checkout -b [new-branch-name] create new branch and switch to it
git add [file-name] add your changes to stage
git add . add all change in current directory
git add .. add all changes one directory up
git commit -m “Commit message" commit staged changes
git commit -v commit and open message in sublime
git push origin [branch-name] push branch to remote repository
git checkout master go back to local master
git fetch origin master bring remote master to machine
git merge origin/master merge remote master to local master
git pull origin master combines fetch and merge
git branch -d [branch-name] delete local branch
git merge master merge master branch into current branch
git clone REPO URL clone the repo to your machine
git reset HEAD [file name] reset to last commit
git reset HEAD^ reset to previous commit
git reset HEAD~3 reset to third commit from HEAD etc
git [command] —help learn about and command
